# AegisShield Discord Bot

Protect your server from raids, spam, and unauthorized actions while keeping essential utilities like ticket systems, moderation, and logging centralized in one reliable bot. Simplify server management and fortify your community with AegisShield!

## Features

### Security & Moderation
- **Honeypot System** - Detects and bans suspicious users who interact with hidden channels
- **Spam Shield** - Automatically detects and deletes spam messages
- **Link Shield** - Blocks unauthorized links with allowlist support
- **Embed Shield** - Controls embed permissions
- **React Shield** - Prevents reaction spam
- **Voice Shield** - Monitors voice channel activity
- **Thread Shield** - Controls thread creation
- **App Shield** - Manages app/activity permissions
- **Poll Shield** - Controls poll creation
- **AI Moderation** - AI-powered content filtering

### Role Management
- **Auto-Role** - Automatically assigns roles to new members
- **Reaction Roles** - Members can self-assign roles by reacting to messages
- **Role Creation** - Create roles with custom colors and permissions

### Welcome & Leave System
- **Welcome Channel** - Set channel for welcome messages
- **Welcome Message** - Customizable with `{user}`, `{username}`, `{server}` variables
- **Leave Channel** - Set channel for leave messages
- **Leave Message** - Customizable message templates

### Ticket System
- **Ticket Setup** - Create ticket channels with custom categories

### Utility Commands
- **Purge** - Bulk delete messages
- **Announce** - Send announcements to channels
- **User Info** - Get detailed user information
- **Server Stats** - Display server statistics
- **Minecraft Status** - Check Minecraft server status
- **Quarantine** - Isolate problematic users
- **Verification** - Set up verification channels

### Fun Commands
- **Cat/Dog Images** - Random animal pictures
- **Polls** - Create interactive polls
